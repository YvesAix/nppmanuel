# Commutateur de document

Le commutateur de document est un panneau de l'interface ou une liste flottante listant les documents ouverts et permettant de naviguer entre eux. Il peut servir d'alternative aux onglets.

Le panneau peut s'activer dans les préférences générales.

Les raccourcis clavier par défaut Ctrl+Tab et Ctrl+Shift+Tab permettent de naviguer entre les documents ouverts au moyen d'un mini-version du commutateur de document qui peut être désactivé ou paramétré dans les préférences diverses.

# NppExport plugin

Ce plugin permet d'exporter votre fichier colorisé en html ou rtf ou de le copier dans le presse papier.

Il a été créé par Harrybharry :

  • Projet sur sourceforge
  • Forum d'aide

# Installation

Il peut être installé ou désinstallé simplement depuis le Plugin manager.

# Utilisation

Choisissez votre option depuis le menu Compléments -> NppExport.

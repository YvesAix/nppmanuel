# Compare

Ce plugin permet de déterminer les différences entre deux fichiers ouverts, entre deux versions Git, SVN ou depuis la dernière sauvegarde.

Il a été créé par Ty Landercasper et Jean-Sébastien Leroy en 2011 et est développé depuis 2016 par Pavel Nedev.

  • https://github.com/jsleroy/compare-plugin
  • Tous les contributeurs

# Installation

Il peut être installé ou désinstallé simplement depuis le Plugin Manager.

# Utilisation

Compare est accessible depuis le menu Compléments -> Compare.

OptionRaccourci clavierDescription
CompareCtrl+Alt+CCompare le document sélectionné avec le document suivant
Set as First to CompareCtrl+Alt+1Définit le premier document à comparer
Clear Active CompareCtrl+Alt+X"Ferme" la comparaison active et remet en ordre la fenêtre
Clear All ComparesCtrl+Alt+Shift+X"Ferme" toutes les comparaisons
Diff since last SaveCtrl+Alt+DCompare le document avec la dernière sauvegarde
SVN diffCtrl+Alt+VCompare le document avec la version courante SVN
Git diffCtrl+Alt+GCompare le document avec la version courante Git

Il affiche alors les deux documents ou versions côte à côte, les synchronise horizontalement et verticalement, affiche un panneau de navigation (NavBar) et indique, par le biais de codes couleurs et puces les éléments ajoutés, supprimés ou modifiés :

# Paramètres

Les paramètres sont accessibles depuis le menu Compléments -> Compare -> Settings.

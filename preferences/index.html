# Préférences

Notepad++ propose un certain nombre d'options depuis la fenêtre Préférences du menu Paramétrage, qui permettent de régler le comportement du logiciel, de ses fonctionnalités et l'aspect de son interface.

  • — Préférences générales
  • — Zone d'édition
  • — Nouveau document
  • — Dossier par défaut
  • — Historique des fichiers récents
  • — Extensions de fichier
  • — Langage
  • — Coloration
  • — Impression
  • — Sauvegarde
  • — Autocomplétion
  • — Instance multiple
  • — Délimiteur
  • — Cloud
  • — Moteur de recherche
  • — Préférences diverses

Les options de configuration sont stockées dans un fichier .xml à la racine des Fichiers de configuration.

%APPDATA%\Notepad++\config.xml
